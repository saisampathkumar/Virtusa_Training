Node JS

  1. npm install npm: For NPM
  2. npm install @Angular/cli: For Angular
  3. npm --v: Check NPM Version
  4. ng --v: Check Angular Version
  5. "package.json" describes package versions
  6. npm outdated: For checking any out dated versions
  7. npm install: For installing latest packages
  8. npm uninstall package_name: For uninstalling packages

Starting Angular

  1. ng new project_name: For creating angular project
  2. ng g --help: To see what we can generate
  3. ng serve: For starting the Developement time web server to build and execute the project and check every changes

    4. Example: ng new Project1

    Note:

    g : generate

    -g : global

Extra Commands

  1. --skip-install: Skipping NPM packages
  2. --routing: Creating routing module
  3. --minimal: Skips some features of angular
  4. --skip-git: No git will be added to the project
  5. --dry-run / -d: Used to check what changes can be made
  6. -s / --inline-style: For skipping CSS files
  7. -S / --skip-test: For skipping test files
  8. -t / --inline-template : For skipping template/HTMl files

    9. Examples:

    ng new Project1 --dry-run

    ng new Project1 -s -S -t

Types Script

  1. TypeScript is Compiled to Java Script by Angular called Transpilation
  2. Uses Classes, Methods, var, let, const
  3. Not yet supported by all browser
  4. TypeScript is super set of JavaScript
  5. Object Oriented Programming
  6. npm install -g typescript: For type script installation

TypeScript Types

  1. var l:string = "" stores string
  2. var m:boolean = true / false stores boolean values
  3. var n: number = 123 stores numbers
  4. var o: any can store any type of data type
  1. Array:can store number of values of specific type
  2. object:stores objects

    3. Example:

    var x:string[] = ["asds", "wasd", "awsdfd"]

    var y:number[] = [1, 2, 3, 8, 9876]

    var z:object[] = []
